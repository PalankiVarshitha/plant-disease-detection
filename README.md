# 🌱 Plant Disease Detection System

A machine learning-based system designed to detect plant diseases from leaf images, enabling early diagnosis and improving agricultural productivity.

## 🚀 Overview
This project focuses on building an intelligent system that analyzes plant leaf images and classifies diseases using machine learning techniques. The goal is to support farmers and agricultural systems in identifying diseases early and taking corrective action.

## ✨ Key Features
- Image-based disease detection  
- Machine learning model for classification  
- Efficient preprocessing and prediction pipeline  
- Lightweight application interface  
- Model experimentation using Jupyter Notebook  

## 🛠 Tech Stack
- Python  
- NumPy, Pandas  
- Scikit-learn  
- OpenCV  
- Flask  
- Jupyter Notebook

## ⚙️ Installation & Setup
1. Clone the repository  
git clone https://github.com/PalankiVarshitha/plant-disease-detection.git  
cd plant-disease-detection  

2. Create virtual environment  
python -m venv venv  

Activate:  
Mac/Linux → source venv/bin/activate  
Windows → venv\Scripts\activate  

3. Install dependencies  
pip install -r requirements.txt  

## ▶️ Run the Application
python app.py  

Open in browser:  
http://localhost:5000
